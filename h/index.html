about

if you want to stalk me, check out the below links.

i can be found on twitter @wowaname, github, keybase, bandcamp, and soundcloud. i also do a lot of stuff on irc and tor.

for other stuff on the site, check out some of people's music i host (including mine), and the files currently being hosted for whatever reason.

irc

i run the irc network volatile (hidden service link) which can be accessed via the roundrobin irc.volatile.club port +6697, +9999, or 6667

or by a specific server listed here (hidden service link). there is also a webchat available for anyone too lazy to set up an irc client.

check the /list for all channels, and if you don't find one you like, /join a new one and invite people there!
